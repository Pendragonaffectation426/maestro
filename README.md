## 🧭 What Maestro Does

Maestro helps you run AI coding agents from one place. It gives you a simple way to manage tasks, pass work between agents, and keep the process clear.

It is built for people who want help with coding work without setting up a complex toolchain. Maestro uses a small core and thin runtime adapters for Claude Code, Gemini, Codex, and Cursor.

## 🛠️ How the Workflow Works

Maestro follows a simple task flow:

1. You describe what you want
2. Maestro breaks the job into steps
3. An agent works on the task
4. Maestro tracks progress
5. You review the result

This helps when a job has many parts. It also makes it easier to move work between agents without losing track of what happened.

## 🔐 Privacy and Local Use

Maestro is built as a portable core with thin runtime adapters. That setup keeps the main logic small and easier to manage.

You stay in control of the tool you connect, the task you send, and the agent you choose. If you use an external AI service, that service may handle the content you send to it under its own rules.

No. It is for anyone who wants to use AI coding agents in a more organized way.

Yes. Maestro supports several agents and lets you switch between them.
